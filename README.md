# addkeysToWindow
给特定得窗口添加快捷键

首先, 你只能设定alt+1 2 3 4 5 6这六个键, 如果想设定更多请去改原代码.


你可以通过**右边的Alt**+数字键 给特定窗口设置一个快捷键

然后用**左边的Alt**+数字键 打开特定的窗口,

如下图所示:
[截图说明](./screenshots.gif)



你也可以通过**右边的Alt**+p键输出目前绑定了哪些窗口
